# ملخص مشروع نظام إدارة المركبات الذكي (Fleet Management System)

المشروع عبارة عن نظام متكامل لإدارة المركبات باستخدام Python (Flask) مع واجهة أمامية HTML/CSS/JavaScript وقاعدة بيانات SQLite.النظام يشمل إدارة الصيانة، الوقود، الحوادث، التنبيهات، واتساب، البريد، بالإضافة إلى لوحة تحكم مركزية وداشبورد رئيسي وشريط جانبي للتنقل.

## المكونات الأساسية للنظام:
- إدارة الصيانة والوقود والحوادث.
- نظام تنبيهات عبر البريد والواتساب.
- لوحة تحكم للتحكم في تشغيل الميزات (Control Panel).
- داشبوردات فرعية للصيانة والوقود والحوادث.
- داشبورد رئيسي يربط كل البيانات في صفحة واحدة.
- أزرار تحكم لتحديث البيانات، إرسال التنبيهات، وتشغيل AutoMonitor.
- شريط جانبي للتنقل بين الصفحات والتحكم في النظام.

## التقنيات المستخدمة:
- **Backend**: Python 3 – Flask
- **Frontend**: HTML – CSS – JavaScript
- **Database**: SQLite
- **Auth**: Google OAuth – Master Password
- **Automation**: VBA – Excel Integration
- **Deployment**: GitHub + Render Hosting
- **Communication**: WhatsApp API – Email SMTP

## حالة المشروع الحالية:
- النظام يعمل على المستضيف بنجاح.
- الكود مرفوع ومنظم داخل GitHub.
- جميع اللوحات والماكروز جاهزة.
- لوحة التحكم والشريط الجانبي مفعّلين.
- النظام جاهز للتطوير أو إضافة مساعد ذكاء اصطناعي.
