## 🔐 2. القواعد الذهبية الخمس (غير قابلة للنقاش)

### القاعدة الأولى: مركزية مصدر البيانات
```
النظام الرقمي هو المصدر الوحيد للحقيقة.
أي بيانات خارج النظام = غير موجودة رسمياً.
```

### القاعدة الثانية: التحديث اللحظي
```
كل تغيير في حالة مركبة أو سائق يُحدَّث خلال 24 ساعة بحد أقصى.
التأخير = إهمال تشغيلي يُدوَّن في سجل المخالفات.
```

### القاعدة الثالثة: التوثيق الإلزامي
```
لا يُقبل أي تعديل يدوي بدون مسوّغ مكتوب في حقل "الملاحظات".
السبب + التاريخ + اسم المسؤول = شروط التعديل المقبول.
```

### القاعدة الرابعة: شفافية التعديل
```
كل تغيير يُسجَّل تلقائياً في سجل التدقيق (Audit Log).
لا يمكن حذف أي سجل بعد إدخاله.
```

### القاعدة الخامسة: المخرجات غير قابلة للتفاوض
```
التقارير الصادرة من النظام هي المرجع الرسمي الوحيد للاجتماعات والقرارات.
تقارير من مصادر أخرى لن تُقبل في أي اجتماع إداري.
```
