<p align="center"><em>Bin Zomah — Fleet Management & Maintenance System</em></p>

---

## الوصف
نظام مُتكامل لإدارة أسطول المركبات يقدّم واجهة بسيطة وميزات عملية لتسجيل المركبات والسائقين، إنشاء الفواتير والتقارير، وإرسال التنبيهات عبر البريد الإلكتروني وWhatsApp. تمّ تصميم الحلّ ليعمل محلياً باستخدام Flask وقاعدة بيانات خفيفة (SQLite) مع إمكانية التصدير إلى ملفات Excel قابلة للطباعة.

## المزايا الرئيسية
- إدارة بيانات المركبات والسائقين (إضافة، تعديل، حذف، بحث ذكي)
- إنشاء فواتير وتقارير بتنسيق Excel (جاهزة للطباعة A4)
- نظام تنبيهات عبر البريد الإلكتروني (SMTP) وWhatsApp API
- لوحة تحكم لعرض إحصائيات وصيانة دورية
- دعم Google OAuth وتوثيق بكلمة سر رئيسية

## التقنيات المستخدمة
- Backend: Python 3.10+، Flask
- Frontend: HTML5, CSS3, JavaScript (Vanilla)
- Database: SQLite
- Excel generation: exceljs (frontend) / openpyxl أو مكتبات Python عند الحاجة

## متطلبات التشغيل (محلي)
1. أنشئ بيئة افتراضية Python
   ```bash
   python -m venv .venv
   source .venv/bin/activate   # على لينكس / ماك
   .venv\Scripts\activate    # على ويندوز
   ```
2. ثبّت المتطلبات:
   ```bash
   pip install -r requirements.txt
   ```
3. أنشئ ملف .env مستندًا إلى `.env.example` واملأ القيم (مثل مفاتيح SMTP وGoogle OAuth وMASTER_PASSWORD).
4. تهيئة قاعدة البيانات (إن لزم):
   ```bash
   python init_db.py
   ```
5. تشغيل التطبيق:
   ```bash
   python app.py
   ```
6. افتح المتصفح وادخل إلى: http://localhost:5000

## ملاحظات عن التشغيل والطباعة
- تم ضبط قوالب تصدير الفاتورة لتكون قابلة للطباعة على ورق بحجم A4. إذا أردت إعدادات مخصصة للطباعة (هوامش أو رأس/تذييل محدّد) أخبرني لأقوم بالتعديل.

## بنية المستودع
- `app.py` — التطبيق الرئيسي (Routes وAPI endpoints)
- `init_db.py` — سكربت مسؤول عن إنشاء قاعدة بيانات SQLite (جدول السائقين والمركبات) ويقوم تلقائياً بقراءة البيانات الأولية من ملف `drivers_data.js` وحفظها في القاعدة إذا كانت فارغة لتسهيل إعداد النظام.
- `templates/` — قوالب HTML
- `static/` — ملفات CSS، JS، والصور
- `database.sqlite` — قاعدة البيانات المحلية (نمطيًا يُنصح بإضافتها إلى .gitignore إن كانت تحتوي بيانات حساسة)

## البريد الإلكتروني (SMTP)
- الإعدادات موجودة في ملف `.env`. تأكد من تفعيل "less secure apps" أو استخدام App Password إذا كنت تستخدم Gmail.
- ملف `patch_email.py` وواجهات إرسال الإيميلات في `templates/index.html` تدعم إرسال الفاتورة كمرفق (Base64).

## أمن وخصوصية
- لا تضع بيانات حساسة (كلمات مرور فعلية أو مفاتيح API) في المستودع العام.
- استخدم ملف `.env` محليّاً وضمّنه في `.gitignore`.
