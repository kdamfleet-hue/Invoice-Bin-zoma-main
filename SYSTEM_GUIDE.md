|--------|---------------|------------------------|

| مركبات أسبير/معطلة | أصول خارج الخدمة | إجراء تشغيلي فوري |

### 3.2 نظام الإنذار المبكر للوثائق

```
🔴 أحمر (انتهت الصلاحية / أيام سالبة):
   → إيقاف تشغيل المركبة/السائق فوراً
   → فتح بلاغ صيانة/تجديد فورياً
   → إشعار مدير الأسطول مباشرة

🟡 أصفر (أقل من 30 يوم):
   → بدء إجراءات التجديد فوراً
   → تسجيل المعالجة في الملاحظات

🟢 أخضر (أكثر من 30 يوم):
   → لا إجراء مطلوب
   → متابعة روتينية أسبوعية
```

## 📅 4. الجدول الأسبوعي – قلب النظام التشغيلي

### 4.1 تعريف الجدول الأسبوعي
الجدول الأسبوعي هو **المرجع التشغيلي الرسمي الوحيد** الذي يضم:
- حالة كل مركبة في الأسطول
- حالة كل سائق
- صلاحية جميع الوثائق
- تنبيهات الإجراءات الفورية المطلوبة

### 4.2 أعمدة الجدول وتعريفاتها

| العمود | التعريف الرسمي | من المسؤول عن تحديثه |
|--------|--------------|---------------------|
| م | رقم تسلسلي تلقائي | النظام |
| الرقم الوظيفي | رقم السائق في السجلات الرسمية | قسم الحركة |
| اسم السائق | الاسم الكامل كما في الإقامة | قسم الحركة |
| رقم الإقامة | رقم إقامة ساري | قسم الحركة |
| الوظيفة | موزع / موصل | قسم الحركة |
| رقم اللوحة | لوحة المركبة الرسمية | قسم الحركة |
| الموديل | سنة الصنع | قسم الأسطول |
| نوع المركبة | التصنيف الرسمي | قسم الأسطول |
| تاريخ انتهاء الفحص الدوري | تاريخ الانتهاء + الأيام المتبقية | قسم الصيانة |
| تاريخ انتهاء رخصة السير | تاريخ الانتهاء + الأيام المتبقية | قسم الحركة |
| تاريخ انتهاء بطاقة التشغيل | تاريخ الانتهاء + الأيام المتبقية | قسم الحركة |
| تاريخ انتهاء بطاقة السائق | تاريخ الانتهاء + الأيام المتبقية | قسم الحركة |
| رقم الجوال | جوال السائق الفعلي | قسم الحركة |
| الملاحظات | أي ملاحظة تشغيلية موثقة | المسؤول المعني |

### 4.3 دورة تحديث الجدول

```
📆 كل أسبوع (الأحد):
   1. استيراد ملف Excel المحدث للنظام
   2. مراجعة جميع التنبيهات الحمراء والصفراء
   3. تحديث الملاحظات لكل حالة استثنائية
   4. تصدير الجدول المُعتمد وإرساله للإدارة

📆 يومياً:
   1. فحص لوحة التنبيهات (5 دقائق)
   2. تحديث أي تغيير طارئ (سائق غياب، عطل مركبة)
```

### 4.4 بروتوكول استيراد Excel

```
الخطوة 1: فتح النظام على تبويب "الجدول الأسبوعي"
الخطوة 2: الضغط على زر "استيراد Excel"
الخطوة 3: اختيار الملف المحدث (يجب أن يكون بنفس البنية المعتمدة)
الخطوة 4: مراجعة البيانات المستوردة قبل الحفظ
الخطوة 5: الضغط على "حفظ وتأكيد"
الخطوة 6: مراجعة التنبيهات الناتجة

⚠️ تحذير: الملفات التي لا تطابق البنية المعتمدة ستُرفض تلقائياً.
```
